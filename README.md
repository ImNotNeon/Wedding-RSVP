# Wedding RSVP – Guest Management System

## Overview

The Wedding RSVP – Guest Management System is a web-based application designed to simplify event confirmation tracking and guest management. The system allows hosts to collect RSVP responses, validate guest entries, and monitor attendance in real time through a connected Google Sheets backend.

This project focuses on creating an efficient, easy-to-deploy solution that requires minimal technical maintenance for clients.

---

## Features

- Online RSVP submission form  
- Guest validation and confirmation tracking  
- Real-time attendance updates  
- Automated data storage in Google Sheets  
- Clean and user-friendly interface  
- Easy deployment for non-technical users  

---

## System Architecture

The application follows a lightweight web-based structure:

- **Frontend:** Handles RSVP form submission and user interaction  
- **Backend (Google Apps Script):** Processes form input, validates guest data, and updates records  
- **Database (Google Sheets):** Stores guest list, confirmation status, and attendance records  


---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- Google Apps Script  
- Google Sheets  

---

## How It Works

1. Guests access the RSVP web form.
2. They submit their name and attendance response.
3. The system validates the input against the guest list.
4. The system asks how many will be going.
5. Confirmation status is recorded in Google Sheets.
6. Event organizers can monitor attendance in real time.

---

## Project Goals

- Simplify RSVP collection and tracking  
- Reduce manual data entry errors  
- Provide real-time visibility of attendance  
- Deliver an accessible and low-maintenance event solution
